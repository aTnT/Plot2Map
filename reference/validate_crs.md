# Validate and standardize CRS across raster files

Validate and standardize CRS across raster files

## Usage

``` r
validate_crs(raster_files, target_crs = "EPSG:4326")
```

## Arguments

- raster_files:

  Vector of raster file paths

- target_crs:

  Target CRS (default: "EPSG:4326")

## Value

List with CRS validation results
